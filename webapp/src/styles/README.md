# Styles

전역 스타일 및 CSS 파일들을 모아둔 폴더입니다.

## 예정된 스타일 파일

- **theme.css**: CSS 변수 정의 (색상, 폰트 등)
- **global.css**: 전역 스타일
- **reset.css**: CSS 리셋 (선택적)
- **animations.css**: 애니메이션 정의

## 색상 테마 (예시)

```css
:root {
  --primary-color: #FF6B9D;
  --secondary-color: #C44569;
  --background-color: #FFF5F7;
  --text-color: #2C3E50;
  --success-color: #52C41A;
  --danger-color: #F5222D;
  --border-color: #E8E8E8;
}
```
