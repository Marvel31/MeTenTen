# Services

비즈니스 로직을 담당하는 서비스 레이어입니다.

## 예정된 서비스

- **AuthService**: 인증 관련 (회원가입, 로그인, 로그아웃, 비밀번호 변경)
- **EncryptionService**: 암호화/복호화 (DEK 관리, AES-256-CBC, PBKDF2)
- **FirebaseDataService**: Firebase 데이터베이스 연동
- **TopicService**: 주제 관리 (CRUD)
- **TenTenService**: TenTen 관리 (CRUD, 암호화/복호화)
- **PartnerService**: 배우자 연결 및 관리
- **FeelingExampleService**: 느낌 표현 예시 관리
- **PrayerService**: 기도문 관리
