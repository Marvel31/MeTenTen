# Types

TypeScript 타입 및 인터페이스 정의 파일들입니다.

## 예정된 타입 파일

- **user.ts**: User, UserProfile 등 사용자 관련 타입
- **topic.ts**: Topic 관련 타입
- **tenten.ts**: TenTen 관련 타입
- **partner.ts**: PartnerInfo 관련 타입
- **feeling.ts**: FeelingExample 관련 타입
- **prayer.ts**: Prayer 관련 타입
- **api.ts**: API 요청/응답 타입
- **common.ts**: 공통 타입 정의
