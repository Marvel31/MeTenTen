# Components

재사용 가능한 UI 컴포넌트들을 모아둔 폴더입니다.

## 예정된 컴포넌트

- Layout: 전체 레이아웃 컴포넌트
- Breadcrumb: 네비게이션 경로 표시
- Loading: 로딩 스피너
- ErrorBoundary: 에러 처리
- Toast: 알림 메시지
- ConfirmDialog: 확인 다이얼로그
- Card: 카드 컴포넌트
- Button: 버튼 컴포넌트
- Timer: 10분 타이머
- TopicModal: 주제 추가/수정 모달
- TenTenModal: TenTen 작성/수정 모달
- InvitePartner: 배우자 초대 모달
- PartnerTenTenView: 배우자 TenTen 읽기 모달
- ChangePassword: 비밀번호 변경 모달
